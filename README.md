# path_finder
***path finder algorithms with visualisation on pygame***

## hotkeys
* **S**: choose start square
* **E**: choose end square
* **W**: draw walls
* **2-9**: draw squares with the appropriate weight
* **RMB**: clear square
* **ARROWS**: change path finding algorithm (at screen caption)

## algorithms
* **BFS**
* **Dijkstra**
* **Greedy**
* **A\***
